# Steinschlag-Projekt

Dieses Projekt simuliert und analysiert Steinschlag-Szenarien.

## Struktur
- `analysis/` – Analyse der Daten, Verteilungen, Fits
- `simulation/` – Skripte zur Simulation neuer Steine
- `data/` – Input- und Beispiel-Daten
- `docs/` – Dokumentation
